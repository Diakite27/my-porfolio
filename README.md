🌐 **[yacouba-diakite.com](https://yacouba-diakite.com)**

## Stack

- **Framework** : Next.js 14 (App Router)
- **Styling** : Tailwind CSS v4, shadcn/ui
- **Typographie** : Space Grotesk + Inter
- **Déploiement** : Vercel
- **SEO** : Open Graph, sitemap.xml, robots.txt
